# num-factor

This library contains some utilities for primality testing and factorization. It builds on `num`'s `BigUint`s type.


## Status

Primality tests:
- [x] Trial division
- [x] Miller-Rabin

Factorization:
- [x] Trial division
- [x] Pollard's rho
- [ ] Quadratic sieve
- [ ] General number sieve
